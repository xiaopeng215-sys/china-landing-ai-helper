## Task 2: Login Experience Fix

### Problem
All auth error messages were in Chinese, making them unreadable for international users.

### Changes Made

**`src/app/api/auth/login/route.ts`**
- `'邮箱和密码不能为空'` → `'Email and password are required.'`
- `'密码长度至少 6 位'` → `'Password must be at least 6 characters.'`
- `'认证服务未配置'` → `'Authentication service is not configured.'`
- `'登录失败，请稍后重试'` → `'Sign in failed. Please try again later.'`

**`src/app/api/auth/register/route.ts`**
- `'请求格式错误'` → `'Invalid request format.'`
- `'注册请求过于频繁，请稍后再试'` → `'Too many registration attempts. Please try again later.'`
- `'邮箱和密码不能为空'` → `'Email and password are required.'`
- `'请输入有效的邮箱地址'` → `'Please enter a valid email address.'`
- `'密码必须至少 8 位...'` → `'Password must be at least 8 characters and contain both letters and numbers.'`
- `'密码过于简单...'` → `'Password is too common. Please choose a stronger password.'`
- `'数据库未配置'` → `'Database is not configured.'`
- `'该邮箱已被注册'` → `'This email is already registered. Try signing in instead.'`
- `'注册失败，请稍后重试'` → `'Registration failed. Please try again later.'`
- `'服务器错误，请稍后重试'` → `'Server error. Please try again later.'`

**`src/app/api/auth/logout/route.ts`**
- `'登出失败'` → `'Sign out failed. Please try again.'`

**`src/app/auth/signup/page.tsx`**

**`src/components/layout/Header.tsx`**

### 1. 修复 AI Prompt 使用中文回复

**文件**: `src/lib/ai-client.ts`

**修改内容**:
- ✅ 更新 `general` Prompt 模板，强化中文回复要求
- ✅ 添加明确的回复策略说明
- ✅ 在结构化 Prompt 中强制要求简体中文回复

**关键改动**:
```typescript
general: `你是 China Landing AI Helper，一位专业的中国旅行助手。

【重要语言要求 - 必须遵守】
- 始终使用简体中文回复，除非用户明确要求使用其他语言
- 严禁在中英文之间混用，保持语言一致性
- 专有名词可保留英文（如 App 名称、品牌名），但句子结构必须是中文
- 数字、价格、时间等使用阿拉伯数字
- 如果用户用中文提问，必须用中文回复

【回复策略】
- 先理解用户需求，再提供建议
- 不要主动推荐景点，等待用户明确询问后再推荐
- 推荐时必须包含图片 URL
- 回复要友好、专业、实用
...`
```

**结构化 Prompt 增强**:
```typescript
const structuredPrompt = options?.structured 
  ? systemPrompt + '\n\n【重要】请用 JSON 格式回复，必须包含以下字段：\n{\n  "text": "你的文字回复 (简体中文)",\n  "recommendations": [{"type": "attraction|restaurant|hotel|transport", "id": "唯一 ID", "name": "名称", "nameEn": "英文名", "reason": "推荐理由", "price": "价格", "location": "位置", "image": "图片 URL (必需)"}],\n  "actions": [{"type": "book|navigate|info", "provider": "klook|trip|amap|didi|meituan", "url": "链接", "text": "按钮文字"}],\n  "images": [{"url": "图片 URL", "caption": "图片说明"}]\n}\n\n【注意】\n- 所有推荐必须包含 image 字段 (图片 URL)\n- 使用 Unsplash 或类似图床的图片链接\n- 文字回复必须使用简体中文'
  : systemPrompt;
```

**影响范围**: 
- ✅ Qwen AI (阿里云百炼)
- ✅ MiniMax AI
- ✅ Mock 回复 (测试/降级)

### 3. 优化推荐逻辑 (先对话后推荐)

**文件**: `src/lib/ai-client.ts`

**修改内容**:
- ✅ 默认回复不再显示"热门景点"推荐
- ✅ 改为引导用户提问，等待明确需求
- ✅ 只在用户询问具体话题时显示相关推荐

**修改前**:
```json
{
  "text": "👋 欢迎来到 China Landing AI Helper!...",
  "recommendations": [
    {
      "type": "attraction",
      "id": "popular-shanghai",
      "name": "热门景点",
      "reason": "探索上海最著名的地标"
    }
  ]
}
```

**修改后**:
```json
{
  "text": "👋 欢迎来到 China Landing AI Helper!\n\n我可以帮你：\n🗽 规划行程 - \"帮我规划上海 4 天行程\"\n🍜 推荐美食 - \"推荐北京美食\"\n🚇 交通指南 - \"怎么坐地铁去外滩？\"\n💳 支付设置 - \"如何设置支付宝？\"\n\n请告诉我你的具体需求，我会为你提供详细、实用的建议！",
  "recommendations": [],
  "actions": [],
  "images": []
}
```

**逻辑流程**:
```
用户消息 → 意图检测 → 匹配场景？
  ├─ 行程规划 → 显示景点推荐 + 图片
  ├─ 美食推荐 → 显示餐厅推荐 + 图片
  ├─ 交通指南 → 显示交通推荐 + 图片
  ├─ 支付设置 → 显示支付指南 + 图片
  └─ 其他/欢迎 → 仅显示引导文字，不推荐
```
